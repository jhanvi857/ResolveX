# ResolveX

Modern DNS Resolver Dashboard built with React, TypeScript, Tailwind CSS, Framer Motion, and Recharts.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Features

- Recursive DNS tracing visualization
- Resolver metrics and query logs
- Analytics charts for latency, cache efficiency, and volume
- Terminal-style live trace output
